FULL TITLE : AN ACT AMENDING THE COOPERATIVE CODE OF THE PHILIPPINES TO BE KNOWN AS THE 'PHILIPPINE COOPERATIVE CODE OF 2002'
SHORT TITLE : Philippine Cooperative Code of 2004
ABSTRACT : "Philippine Cooperative Code of 2004." The measure seeks to amend the Cooperative Code to make the organization of cooperatives more attractive and to provide a fertile social and economic environment for cooperatives to succeed in their endeavors.
